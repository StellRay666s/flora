FLORA STUDIO
Сервис представляет из себя цветочный интернет магазин с воможностью авторизации/регистрации, добавления товара в корзину и оформления заказа.

Demo: https://flora-35btt9y7o-stellray666s.vercel.app/.

Tech Stack(frontend):

React.js,
SCSS(css-modules)
ReduxToolKit
Axios,
React-Router-Dom,
